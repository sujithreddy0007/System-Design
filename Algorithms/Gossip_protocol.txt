Gossip Protocol
Definition

Gossip Protocol is a communication algorithm where nodes periodically exchange information with a few random nodes, causing information to spread throughout the system similar to how rumors spread among people.

How It Works
A node receives new information.
It randomly selects a few peers.
It shares the information with them.
Those peers repeat the process.
